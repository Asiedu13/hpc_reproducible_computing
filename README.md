# hpc_reproducible_computing
## Project Description
This program takes in any integer (or number) and squares it

## How to run
1. Install python from (https://www.python.org/downloads/)[python]
2. Download the analysis.py file from this project
3. Run program with any number of your choice
```python 
python analysis.py 5
 ```

### Results
The following should be displayed for the following inputs
 ```python 
python analysis.py 5
Input: 5
Squared: 25

python analysis.py 12
Input: 12
Squared: 144
 ```

## Reproducibility
Reproducibility is the system that ensures honest continuous research and progress.
